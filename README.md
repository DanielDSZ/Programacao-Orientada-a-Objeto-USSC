# Programacao-Orientada-a-Objeto-USSC

Este repositório contém exemplos de código Java que abrangem diversos conceitos, desde a sintaxe básica até a implementação de módulos, orientação a objetos (OO), associação e herança de classes.

## Conteúdo

1. **Sintaxe Básica**
   - Exemplos simples para entender a sintaxe fundamental da linguagem Java.

2. **Módulos**
   - Demonstração de como organizar e estruturar seu código usando módulos.

3. **Orientação a Objetos (OO)**
   - Implementação de classes e objetos para entender os princípios da OO.

4. **Associação de Classes**
   - Exemplos de como classes podem estar associadas entre si.

5. **Herança de Classes**
   - Demonstração de como utilizar a herança para reutilizar e estender funcionalidades.
